### Installation

[Node.js](https://nodejs.org/) v6+ to run.
[Mongo DB](https://www.mongodb.com/)

Install the dependencies and devDependencies and check and start the server.

```sh
$ node -v
exit v8.x
$ nodemon -v
exit 1.x
```
